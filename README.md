Информационная система учета партий молочной продукции

АО ГК «Российское молоко»

 Описание проекта

Данное приложение представляет собой информационную систему для учета партий молочной продукции. Программа разработана с использованием технологии WPF на языке программирования C# и предназначена для работы с базой данных SQL Server.

Система позволяет хранить, просматривать и добавлять информацию о продукции, складах и партиях готовой продукции.

 Цель разработки

Создание простого программного решения для автоматизации учета движения и хранения молочной продукции на складе предприятия.

 Функциональные возможности

Программа реализует следующие функции:

просмотр списка партий продукции
отображение информации о продукции и складах
добавление новых партий продукции
хранение данных о сроках годности
поиск партий по названию продукции
 Структура базы данных

В системе используется база данных «РоссияМолоко», содержащая следующие таблицы:

 Продукция

Содержит информацию о видах продукции:

название
жирность
тип упаковки
 Склады

Содержит информацию о складах:

название склада
адрес
 Партии

Содержит информацию о партиях продукции:

ссылка на продукцию
ссылка на склад
дата производства
срок годности
количество
 Используемые технологии
Язык программирования: C#
Технология интерфейса: WPF
СУБД: SQL Server
Среда разработки: Visual Studio
Система контроля версий: Git
 Запуск приложения

Для запуска программы необходимо:

Создать базу данных с помощью файла database_script.sql в SQL Server Management Studio (SSMS)
Открыть проект в Visual Studio
Проверить строку подключения к базе данных в файле DbHelper.cs
Запустить приложение
 Назначение программы

Программа может использоваться в учебных целях для демонстрации работы клиент-серверных приложений, а также в качестве прототипа системы учета продукции на предприятии молочной промышленности.
